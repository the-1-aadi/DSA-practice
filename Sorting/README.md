## Sorting

### Selection_sort.cpp
sorting by swapping the smallest element with the first element
in the array.
TC: O(n^2) -> best, average, and worst case
SC: O(1) -> Uses constant memory for swapping elements

### Bubble_sort.cpp
sorting by swapping adjacent terms and moving the largest term to the end
// TIME: O(n^2)-> avg., worst case, O(n)-> best case
// SPACE: O(1)
